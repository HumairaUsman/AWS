# Deploy a WordPress blog on AWS Lightsail.
Deploying a wordpress on AWS lightsail and configuring static IP and backing up the instance.

## Prerequisites:
✅ An AWS account

# STEPS
:one: Sign in to AWS and Navigate to Lightsail
2️⃣ Choose an Instance Location
3️⃣ Select an Instance Image
4️⃣ Choose an Instance Plan
5️⃣ Name and Create the Instance
6️⃣ Configure static IP
7️⃣ Retrieve WordPress Admin Credentials
